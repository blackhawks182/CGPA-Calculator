# In IUT, CGPA Matters
A modular C project for calculating CGPA.

Build (Linux):
```
chmod +x make.sh
./make.sh
```
Build (Windows):
```
.\make.ps1
```
Test (Linux):
```
chmod +x test.sh
./test.sh
```
Test (Windows):
```
.\test.ps1
```
Run (Linux):
```
chmod +x calculator.out
./calculator.out
```
Run (Windows):
```
.\calculator.exe
```
